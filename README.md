# PalavrasPrimas
### Desafio
O desafio consiste em descobrir se a somatória dos valores das letras da palavra é um número primo.

    Regras:
    1- A palavra deve conter apenas letras maiúsculas e minúsculas, sem espaços.
    2- As letras minúsculas (a - z) valem ( 1 - 26) respectivamente
    3- As letras maiúsculas (A - Z) valem (27 - 52) respectivamente

Depois de conseguir todos os valores das letras e calcular a soma, verificamos se o resultado da soma corresponde a um número primo.
